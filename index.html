HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements nested inside other elements are known as child elements.
  • Semantic elements use names which describe the content and role an element plays on a webpage.
  • Content is contained inside the opening and closing tags of an element.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A period designates a class selector
  • To link an external CSS style sheet to the HTML use a link element with the attributes rel and href nested inside
  • colour indicates the colour of the text

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches back to main branch
  • git pull origin main: pulls down the latest main branch
  • git add -A: adds all changes in the current working branch to the staging area
  • git commit -m "commit message": a commit was created that shows the current staged changes
  • git push origin: pushes the changes to the remote branch
